# yeyTaken/hytale/language/v/2026.01.24-6e2d4fc36

Pacote de linguagem para **Hytale**.

## Conteúdo

Arquivos incluídos neste pacote:

- `pt-BR/*`
- `fallback.lang`

## Instalação

### Linux (Flatpak)

Copie os arquivos para o diretório de idiomas do Hytale:

```bash
cp -r pt-BR fallback.lang ~/.var/app/com.hypixel.HytaleLauncher/data/Hytale/install/release/package/game/latest/Client/Data/Shared/Language/
````

### Windows

1. Abra o **Explorador de Arquivos**.
2. Navegue até a pasta de instalação do Hytale, normalmente em algo como:

```
%AppData%\Hytale\install\release\package\game\latest\Client\Data\Shared\Language\
```

3. Cole os arquivos `pt-BR` e `fallback.lang` dentro dessa pasta.

## Observações importantes

* Atualizações do launcher podem sobrescrever os arquivos instalados.
* Certifique-se de estar usando a versão correta dos arquivos para a versão do jogo.
